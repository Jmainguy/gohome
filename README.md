Dynamic DNS using nsd and ssh

## Usage
Run against a bind file.

```/bin/bash
gohome /tmp/soh.re
```

## PreBuilt Binaries
Grab Binaries from [The Releases Page](https://github.com/Jmainguy/gohome/releases)

## Build
```/bin/bash
go build
```
